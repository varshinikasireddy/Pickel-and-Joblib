# Machine Learning with Python: Save and Load Trained Model

## Objective
This project demonstrates how to train a Linear Regression model using Scikit-Learn and save the trained model for future use using:

- Pickle
- Joblib

## Dataset
The dataset contains:

- Area of house (square feet)
- House price

## Steps Performed
1. Load dataset using Pandas
2. Train Linear Regression model
3. Predict house price
4. Save model using Pickle
5. Load saved model
6. Save model using Joblib
7. Load saved model
8. Verify predictions from loaded models

## Why Save Models?

Training a model repeatedly is time-consuming. Saving the trained model allows us to:

- Reuse the model instantly
- Deploy it in production applications
- Build APIs and web applications
- Reduce computation time

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Pickle
- Joblib
- Jupyter Notebook
